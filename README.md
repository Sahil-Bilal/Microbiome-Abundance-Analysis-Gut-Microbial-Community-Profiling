## Overview
This project contains a genus-level gut microbiome abundance table (simulated to match real dataset characteristics) and a Jupyter notebook demonstrating EDA and visualization workflows commonly used in microbiome research.

## Files
- Microbiome_genus_project.ipynb — Notebook with embedded visualization links.  
- genus_counts_realstyle.csv — Raw counts (samples × genera).  
- genus_rel_abundance_realstyle.csv — Relative abundances.  
- genus_alpha_metrics.csv — Shannon & Simpson metrics.  
- genus_clr_matrix.csv — CLR-transformed matrix.  
- genus_pca_coords.csv — PCA coordinates.  
- library_size_hist.png, shannon_hist.png, top15_genera.png, pca_clr.png, corr_clr.png — Visuals included.

## How to run
1. Clone or download the repo and ensure all files are in the same directory.  
2. Install dependencies:
```
pip install pandas numpy matplotlib seaborn scikit-learn
```
3. Open the notebook in Jupyter and run cells to recompute or inspect outputs.
